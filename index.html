    V3

    Clouds and weather feel

    Uses recent NASA GIBS true-colour imagery as a cloud source and IMERG precipitation imagery for active rain/snow areas.

    V4

    Seasons and orbital view

    Computes solar direction, axial tilt and day-of-year geometry in the browser so users can speed up the seasonal cycle.

    V5

    Ocean currents

    Adds GHRSST MUR sea-surface temperature from NASA GIBS and animated current particles informed by major circulation pathways.

    V5

    Human activity

    Blends NASA Black Marble on the night side and adds subtle pulses over major activity hubs for a living human layer.

    Data source credits

    Base Earth: NASA Blue Marble Next Generation via NASA Global Imagery Browse Services (GIBS). Night-side human lights: NASA VIIRS Black Marble via GIBS. Clouds/weather feel: recent VIIRS/MODIS corrected reflectance imagery via GIBS. Precipitation: NASA GPM IMERG precipitation-rate imagery via Worldview/GIBS. Ocean temperature: GHRSST MUR sea-surface temperature via GIBS. Ocean-current movement: lightweight visual streamlines informed by NASA ECCO and OSCAR ocean-current datasets; this browser-only MVP does not download heavy NetCDF current-vector files.

    The app fetches visual layers at page load. If a near-real-time layer is unavailable for the selected recent timestamp, the app falls back to the next available visual layer or keeps the procedural visual.
